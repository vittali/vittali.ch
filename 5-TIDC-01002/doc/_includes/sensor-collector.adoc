=== Task 0

Since I want to connect the two launchpads simultaneously, I need to give each
launchpad.footnote:[more preciously, the debug probe included on the launchpad]
a unique serial number.

==== Setting the serial number of the collector board

I label the collector board (see <<hardware>>) , verify that it is the only board connected to the host
 and set the serial number to `1`:

[source, bash]
----
cd $CCS_INSTALLDIR/ccs910/ccs/ccs_base/common/uscif/xds110
$ ./xdsdfu -m  <1>
$ ./xdsdfu -s 1 -r <2>
----
<1> set the device into DFU mode (required)
<2> set the serial number to _1_ and reset the device


==== Setting the serial number of the sensor board

I label the sensor board (see <<hardware>>) , verify that it is the only board connected to the host
 and set the serial number to `2`:

[source, bash]
----
cd $CCS_INSTALLDIR/ccs910/ccs/ccs_base/common/uscif/xds110
$ ./xdsdfu -m  <1>
$ ./xdsdfu -s 2 -r <2>
----
<1> set the device into DFU mode (required)
<2> set the serial number to _2_ and reset the device
